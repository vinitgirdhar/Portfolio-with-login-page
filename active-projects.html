Fitness Tracker App

A fitness tracking application built using Java Swing and JFreeChart, allowing users to log workouts, adjust weights and repetitions, and visualize their progress through interactive graphs.

Key Features

  • Dynamic Input: Users can input weights and repetitions with the ability to increment/decrement values.
  • Workout History: Users can view a history of all recorded workouts.
  • Graphical Representation: Visualizes workout progress using interactive charts.

Technologies Used

  • Languages: Java
  • Frameworks: Swing for GUI, JFreeChart for graphing
  • IDE: IntelliJ IDEA
